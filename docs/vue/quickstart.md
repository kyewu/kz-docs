### create project

```sh
$ pnpm create vue@latest
```

### Import css library
1. Add unocss with vite
```sh
$ pnpm add -D unocss
```
2. The Tailwind CSS / Windi CSS compact preset for UnoCSS.
```sh
$ pnpm add -D @unocss/preset-wind
```
3. add icons
```sh
$ pnpm add -D @unocss/preset-icons @iconify/json
```
